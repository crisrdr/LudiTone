# Blockly, Web Audio API and ready-made Blocks for music teaching with code.

This repo shows an on-going implementation of Blockly-based platform for music programming using Tone.js as underlying javascript library.
A previous project with this same technology is Toneblocks (https://nime.org/proc/nime21_45/index.html) made by Michael Quigley, and William Payne.
This repo was used at the ICLC2025 with the following purpose: to approach a different way to teach computer music with code, where teachers as well as students can create their own music blocks.
